# Phase 1 Navigation Structure

## Primary Navigation
- Dashboard
- Members
- Check-in
- Payments
- Plans
- Passes
- Reports
- Branches
- Staff
- Settings

## Default Navigation Order
This order is optimized for daily usage frequency:
1. Dashboard
2. Members
3. Check-in
4. Payments
5. Plans
6. Passes
7. Reports
8. Branches
9. Staff
10. Settings

## Role-Based Navigation

### Gym Owner
- Dashboard
- Members
- Check-in
- Payments
- Plans
- Passes
- Reports
- Branches
- Staff
- Settings

### Staff
- Dashboard
- Members
- Check-in
- Payments
- optional Reports if granted
- hide Branches, Staff, and most Settings by default

## Secondary Navigation

### Members
- Members list
- Member detail
- Membership history
- Pass history
- Attendance history
- Payment history

### Settings
- Workspace profile
- Branding
- Branch settings
- QR settings
- Permissions
- Audit logs

## Global Actions
- global member search
- branch switcher or branch filter
- create member
- record payment
- open check-in

## Mobile Navigation Pattern
- top app bar with page title and contextual actions
- bottom priority actions optional later, not required in MVP
- drawer for full section navigation
- preserve branch context visibility at all times
