# Phase 1 Screen Specs

## Login
- email and password fields
- forgot password link
- invite acceptance compatibility
- clear auth error handling

## Onboarding
- workspace name and slug
- first branch form
- gym profile basics
- success confirmation and redirect to dashboard

## Dashboard
- active members
- expiring soon
- today's check-ins
- today's revenue
- recent payments
- recent activity

## Members List
- search by name, email, phone, member code
- filters by branch, status, plan, archive state
- create member action
- quick access to member detail

## Member Detail
- member profile header
- assigned memberships
- assigned passes
- payment history
- attendance history
- notes and emergency contact
- quick actions: sell, renew, check in, edit

## Plans
- plan list
- create/edit plan
- plan price, validity, branch access, rules

## Passes
- pass list
- create/edit pass
- pass validity, usage limits, branch access

## Check-In
- QR scanner area
- manual search field
- validation result card
- failed reason display
- recent check-in feed

## Payments
- payment entry form
- link to member and product
- payment history list
- receipt reference support

## Reports
- date range filter
- active members
- expired members
- check-in summaries
- revenue summaries
- branch performance

## Settings
- workspace profile
- branding
- staff permissions overview
- branch settings
- QR settings
- audit log access
