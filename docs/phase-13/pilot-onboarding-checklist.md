# Pilot Onboarding Checklist

## For Each Pilot Gym
- create or verify owner account
- confirm workspace exists
- confirm first branch exists
- confirm at least one active membership plan exists
- add at least three test members
- record one payment
- execute one successful check-in
- execute one failed check-in

## Owner Training
- log in and sign out
- invite staff
- update workspace settings
- create and edit a member
- assign and renew a membership
- view dashboard metrics

## Staff Training
- search members
- execute manual check-in
- understand failed validation reasons
- record manual payment

## Feedback To Capture
- slowest workflow
- confusing screen labels
- missing fields in member flow
- missing fields in payment flow
- check-in validation errors that need better explanation
